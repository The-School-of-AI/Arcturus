# Capstone Index

This folder contains the execution system for capstone delivery.

## Core Documents
- `MASTER_20_DAY_PLAN.md` - 20-day execution cadence and quality gates.
- `STUDENT_TEAM_ASSIGNMENTS_TEMPLATE.md` - assignment sheet template.
- `PR_REVIEW_OPERATING_FLOW.md` - instructor daily PR review workflow.
- `REPO_STRUCTURE_AUDIT.md` - cleanup status and hygiene recommendations.
- `GITHUB_SETUP_RUNBOOK.md` - GitHub setup steps.
- `BRANCH_PROTECTION_REQUIRED_CHECKS.md` - required check names and constraints.

## Project Charters
- `project_charters/` contains P01-P15 detailed charters.
- Each charter includes:
  - execution addendum,
  - mandatory test gate,
  - 10 hard gate conditions,
  - required delivery README contract.

## Per-Project Delivery Files
- Template: `project_charters/DELIVERY_README_TEMPLATE.md`
- Student/project submissions: `project_charters/PXX_DELIVERY_README.md`

## CI Mapping
- Project gate mapping: `project_charters/PROJECT_GATE_MANIFEST.md`
- Check name source: `../ci/project_required_checks.txt`
